How to start the project:

Backend:
1. Open terminal in the flask folder
2. python -m venv venv
3. venv\Scripts\activate (windows)
4. pip install -r requirement.txt
5. flask run

Frontend
1. Open terminal in the React folder
2. npm install
3. npm run dev

- Make sure Flask backend is running on http://localhost:5000
- React frontend runs on http://localhost:5173
